# I-JEPA-Specific Training Options

{% code fullWidth="true" %}
```
patch_size: Size of the patches to be masked/predicted during training.

embed_dim: Embedding dimension size.

encoder_heads: Number of encoder heads.

encoder_depth: Number of encoder layers.

decoder_depth: Number of decoder layers.

target_aspect_ratio: Aspect ratio of target blocks.

target_scale: Scaling factor for target blocks.
 
context_aspect_ratio: Aspect ratio for context block.

context_scale: Scaling factor for context block.

number_target_blocks: Number of target blocks used in training.

```
